基础命令

查看hugo版本

hugo version

创建一个hugo项目

hugo new site mysite

创建一个页面

hugo new post/first.md

本地调试，在浏览器输入http://localhost:1313/访问

hugo server --buildDrafts

部署到github pages

  1. 在github上创建一个新的仓库（注：仓库名为username.github.io）

  2. 在blog根目录的git上输入huge来生成public文件夹

  3. 进入public文件夹

    cd public
git init
git add .
git commit -m "frist"
git remote add origin https://github.com/a390177226/a390177226.github.io.git
git push -u origin master

  4. 在浏览器输入https://a390177226.github.io/访问

自动部署脚本

Hugo部署方案 - 神秘极客 (xbc.me)

建立deploy.sh文件，输入sh deploy.sh执行脚本

#!/bin/bash
# Program:
#   发布Hugo生成文件到GitHub Pages
# History:
# 2020.05.05 GeekWho First release.
# If a command fails then the deploy stops
set -e

printf "\033[0;32mDeploying updates to GitHub...\033[0m\n"

# Build the project.
Hugo --minify # if using a theme, replace with `Hugo -t <YOURTHEME>`

# Go To Public folder
cd public

# Add changes to git.
git add .

# Commit changes.
msg="Published on $(date +'%Y-%m-%d %H:%M:%S')"
if [ -n "$*" ]; then
    msg="$*"
fi
git commit -m "$msg"

git pull --rebase origin master
# Push source and build repos.
git push -f origin master
