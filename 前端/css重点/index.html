CSS Mindmap

什么是 CSS？

  • CSS 指的是层叠样式表(Cascading Style Sheets)
  • CSS 描述了如何在屏幕、纸张或其他媒体上显示 HTML 元素
  • CSS 也是一种标记语言，用于定义网页的样式，包括针对不同设备和屏幕尺寸的设计和布局
  • CSS 节省了大量工作。它可以同时控制多张网页的布局
  • 外部样式表存储在 CSS 文件中，只需改一个文件即可更改整个网站的外观

如何添加 CSS

方式 位置 特点
Inline CSS 在相关元素的 “style” 属性中定义 书写方便，权重高
Internal CSS 在 HTML 页面<head> 部分内的 <style> 元素中定义 控制一张 HTML 页面的样式
External CSS 在 HTML 页面 <head> 部分内的 <link> 元素中定义 控制多个 HTML 页面的样式 
<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="mystyle.css">

<style>
    p {color:blue}
</style>    
</head>
<body>

<h1 style='color:red'>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

“mystyle.css”：

body {
  background-color: lightblue;
}

CSS语法

CSS 语法由选择器和声明块组成：

  • 选择器指向需要设置样式的 HTML 元素
  • 声明块用花括号括起来，包含一条或多条用分号分隔的声明
  • 每条声明都包含一个 CSS 属性名称和一个值，以冒号分隔

Selectors

CSS 选择器用于“查找”（或选取）要设置样式的 HTML 元素。

我们可以将 CSS 选择器分为五类：

  • 简单选择器（根据名称、id、类来选取元素）
  • 组合器选择器（根据它们之间的特定关系来选取元素）
  • 伪类选择器（根据特定状态选取元素）
  • 伪元素选择器（选取元素的一部分并设置其样式）
  • 属性选择器（根据属性或属性值来选取元素）
Simple selectors Example Example description
#id #firstname Selects the element with id=“firstname”
class .intro Selects all elements with class=“intro”
element.class p.intro Selects only <p> elements with class=“intro”
* * Selects all elements
element p Selects all <p> elements
element,element,… div, p Selects all <div> elements and all <p> elements

Box Model

盒模型实质上是一个包围每个 HTML 元素的盒子。它包括：外边距、边框、内边距以及实际的内容。

标准盒模型

在标准模型中，如果你给盒设置 width 和 height，实际设置的是 content。

padding 和 border 再加上设置的宽高一起决定整个盒子的大小。

替代(IE)盒模型

通过设置box-sizing: border-box;使用IE盒模型，所有宽度都是可见宽度，所以内容宽度是该宽度减去边框和填充部分。

Cascade, Specificity, Inheritance

层叠、优先级和继承，决定着如何将CSS应用到HTML中，以及如何解决冲突

  • 层叠顺序，重要性排序
    1. !important可以用来覆盖所有所有优先级计算
    2. 当应用两条不同级别的规则到一个元素的时候，会根据优先级来选择
    3. 当应用两条同级别的规则到一个元素的时候，后面的规则覆盖前面的规则
  • 优先级
选择器 千位 百位 十位 个位 优先级
元素、伪元素选择器 0 0 0 0 0001
类、属性、伪类选择器 0 0 1 0 0010
id选择器 0 1 0 0 0100
内联样式 1 0 0 0 1000

Note: 在进行计算时不允许进行进位，例如，20 个类选择器仅仅意味着 20 个十位，而不能视为两个百位，也就是说，无论多少个类选择器的权重叠加，都不会超过一个 ID 选择器。

  • 继承：需要在上下文中去理解
    • 一些设置在父元素上的css属性是可以被子元素继承的，有些则不能

Layouts

Display, float, position

Display float position
用于控制布局的最重要的 CSS 属性 用于定位和格式化内容 用于让盒子自由的在某个盒子内移动位置或者固定屏幕中某个位置，并且可以压住其他盒子
规定是否/如何显示元素 最典型的应用：可以让多个块级元素在一行内排列显示 定位=定位模式+边偏移（top、bottom、left 和 right）
大多数元素的默认 display 值为 block 或 inline 为了避免元素浮动后周围的元素会重新排列，使用 clear 属性清除浮动 子绝父相比较常见；z-index 属性指定元素的叠放顺序

Flexbox, grid

长久以来，CSS 布局中唯一可靠且跨浏览器兼容的创建工具只有 floats 和 positioning。这两个工具大部分情况下都很好使，但是在某些方面它们具有一定的局限性，让人难以完成任务。

Flexbox grid
用于按行或按列布局元素的一维布局方法 用于web的二维布局系统
可以更轻松地设计灵活的响应式布局结构，而无需使用浮动或定位 非常轻松地实现一些复杂的布局
