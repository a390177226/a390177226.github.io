What is HTML?

HTML 是用来描述网页的一种语言。

  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)

  • HTML 不是一种编程语言，而是一种标记语言 (markup language)

  • 标记语言是一套标记标签 (markup tag)

  • HTML 使用标记标签来告知浏览器如何组织页面，即描述页面的结构

Basic Document

<!DOCTYPE html>
<html>
<head>
    <title>Document</title>
</head>
<body>
    <h1>My First Heading</h1>
	<p>My first paragraph.</p>   
</body>
</html>
  • <!DOCTYPE html>告诉浏览器文档是HTML5版本编写，它不是 HTML 标签, 只是文档类型声明
  • <html></html>是页面中最大的标签，称为根标签
  • <head></head>定义文档的头部，包含了所有想包含在HTML页面中但不想在HTML页面中显示的内容
  • <body></body>定义文档的主体，包含了访问页面时所有显示在页面上的内容，文本，图片，游戏等等

Tags, Attributes, and Elements

Tags

HTML 标记标签通常被称为 HTML 标签 (HTML tag)。

  • HTML 标签是由尖括号包围的关键词，比如 <html>
  • HTML 标签通常是成对出现的，我们称为双标签，比如 <b>和 </b>
  • 标签对中的第一个标签是开始标签（opening tag），第二个标签是结束标签（closing tag）
  • 有些特殊的标签必须是单个标签（极少情况），例如<br />，我们称为单标签
  • HTML 标签大小写不敏感，但W3C推荐小写

Attributes

HTML 标签可以拥有属性。属性提供了有关 HTML 元素的更多的信息。

属性总是以名称/值对的形式出现，比如：name=“value”。

属性总是在 HTML 元素的开始标签中规定。

Elements

HTML 元素指的是从开始标签到结束标签的所有代码。

  • HTML 元素以开始标签起始,以结束标签终止,元素的内容是开始标签与结束标签之间的内容
  • 某些 HTML 元素具有空内容（empty content），例如<br>
  • 空元素在开始标签中进行关闭（以开始标签的结束而结束）
  • HTML 元素可以嵌套

Basic Elements

Headings

HTML 标题是要在网页上显示的标题或副标题。

  • HTML 标题总共有六个<h1>-<h6>
  • <h1> 定义最大的标题，<h6> 定义最小的标题
  • HTML 标题对于显示文档结构很重要

Paragraphs

段落总是从新行开始，通常是一个文本块。

  • 段落是通过 <p> 标签定义的

  • 换行使用<br>标签

  • 所有连续的空格或空行都会被算作一个空格

Links

几乎可以在所有的网页中找到链接。点击链接可以从一张页面跳转到另一张页面。

  • 链接的基本语法是：<a href="url">link text</a>
  • href属性指定了链接目标的 URL
  • 如果设置target="_blank"属性，则会在新窗口中打开该链接
  • “link text” 不必一定是文本，图片或其他 HTML 元素都可以成为链接

链接也可以实现页面内部跳转功能

  1. 使用id属性创建书签：<h2 id="C4">Chapter 4</h2>
  2. 添加指向书签的链接：<a href="#C4">Jump to Chapter 4</a>
  3. 还可以在另一页上添加指向书签的链接：<a href="html_demo.html#C4">Jump to Chapter 4</a>

Images

在 HTML 中，图像由 <img> 标签定义，它并不会在网页中插入图像，而是从网页上链接图像。

  • 语法：<img src="url" alt="alternatetext">
  • src属性指定图像的路径，必需属性
  • alt属性指定图像无法显示时的备用文本，必需属性
  • width和height设置图片大小，不推荐使用，使用CSS样式代替

Lists

HTML 支持有序、无序和自定义列表

列表语法特点
无序列表无序列表始于 <ul> 标签，每个列表项始于 <li>使用最多
有序列表有序列表始于 <ol> 标签，每个列表项始于 <li>使用较少
自定义列表自定义列表以 <dl> 标签开始，每个列表项以 <dt> 开始，每个列表项的定义以 <dd> 开始使用最少

Tables

表格由 <table> 标签来定义，主要用于显示、展示数据。

表格的表头由 <th> 标签定义，表格的每一行由 <tr> 标签定义。

表格的每一个单元格由 <td> 标签进行定义，单元格可以包含文本、图片、列表等等。

Forms

表单用于收集用户输入的数据。例如，它们可以用作web应用程序的接口，或者通过web发送数据。

表单由表单域、表单元素、提示信息组成。

<form>标签定义了表单域，即把它范围内的表单元素信息提交给服务器。

常用的表单元素：

  • 最重要的是<input> 元素，类型很多，由type属性指定
  • <select> 元素定义下拉列表，<option>元素定义待选择的选项
  • <textarea> 元素定义多行输入字段（文本域）
  • <button>元素定义按钮

<input> 元素的输入类型

  • <input type="text"> 定义单行文本输入字段
  • <input type="password"> 定义密码字段
  • <input type="radio"> 定义单选按钮
  • <input type="checkbox"> 定义复选框
  • <input type="submit"> 定义提交表单数据至表单处理程序的按钮
  • <input type="button> 定义按钮

Others

Comments

HTML 注释不会显示在浏览器中，但它们可以帮助记录HTML源代码。

<!-- Write your comments here -->

Entities

HTML 中的预留字符必须被替换为字符实体。

例如，在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签。

常用描述实体名称
空格符&nbsp;
<less-than&lt;
>greater-than&gt;
&ampersand&amp;
©版权（copyright）&copy;
²superscript 2&sup2;

如需完整的实体符号参考，请访问 HTML 实体符号参考手册 (w3school.com.cn)

Text Formatting

HTML 可定义很多供格式化输出的元素，比如粗体和斜体字。

常用标签描述
<b></b>或<strong></strong>加粗
<i></i>或<em></em>倾斜
<del></del>删除线
<ins></ins>下划线

如需完整的格式化标签，请访问HTML 文本格式化 (w3school.com.cn)
