HTML Mindmap

What is HTML?

HTML 是用来描述网页的一种语言。

  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)

  • HTML 不是一种编程语言，而是一种标记语言 (markup language)

  • 标记语言是一套标记标签 (markup tag)

  • HTML 使用标记标签来告知浏览器如何组织页面，即描述页面的结构

What is HTML5?

HTML5 是 最新的 HTML 标准，由W3C在 2014 年完成标准制定

  • HTML5 是专门为承载丰富的 web 内容而设计的，并且无需额外插件
  • HTML5 拥有新的语义、图形以及多媒体元素
  • HTML5 提供的新元素和新的 API 简化了 web 应用程序的搭建
  • HTML5 是跨平台的，被设计为在不同类型的硬件（PC、平板、手机等）之上运行

Basic Document

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of the document</title>
</head>

<body>
Content of the document......
</body>

</html>
  • <!DOCTYPE html>告诉浏览器文档是HTML5版本编写，它不是 HTML 标签, 只是文档类型声明
  • <html></html>是页面中最大的标签，称为根标签
  • <head></head>定义文档的头部，包含了所有想包含在HTML页面中但不想在HTML页面中显示的内容
  • <body></body>定义文档的主体，包含了访问页面时所有显示在页面上的内容，文本，图片，游戏等等

Tags, Attributes, and Elements

Tags Elements Attributes
由尖括号包围的关键词，比如 <html> 以开始标签起始,以结束标签终止,元素的内容是二者之间的内容 所有HTML元素都可以有属性
通常是成对出现的，我们称为双标签，比如： <b>和 </b> 某些 HTML 元素具有空内容（empty content），例如<br/> 提供了有关 HTML 元素的更多的信息
双标签中的第一个是开始标签，第二个是结束标签 空元素在开始标签中进行关闭（以开始标签的结束而结束） 总是以名称/值对的形式出现，比如：name=“value”
有些标签是单标签（极少情况），例如<br /> HTML 元素可以嵌套 总是在 HTML 元素的开始标签中规定

Block and Inline Elements

每个HTML元素都有一个默认的显示值，这取决于它是什么类型的元素。

Display value 区别 例子
Block-level elements 总是从新行开始，占据可用的全部宽度，有上下外边距（margin） <p>
Inline elements 不会从新行开始，只占用所需的宽度，没有上下外边距（margin） <a>
