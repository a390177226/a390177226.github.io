特殊字符

在 HTML 中，某些字符是预留的。不能使用包含这些字符的文本。

在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签。

显示结果描述实体名称
空格符&nbsp;
<less-than&lt;
>greater-than&gt;
&ampersand&amp;
©版权（copyright）&copy;
²superscript 2&sup2;

如需完整的实体符号参考，请访问我们的 HTML 实体符号参考手册。

骨架标签

每个网页都会有的基本的结构标签。

标签名定义说明
<html></html>HTML标签页面中最大的标签，我们称为根标签
<head></head>文档的头部注意在head标签中我们必须要设置的标签是title
<title></title>文档的标题网页的标题
<body></body>文档的主体页面内容基本上是放在body里的

VSCode工具生成骨架标签代码

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

相较之前，主要多了：

  • <!DOCTYPE html>告诉浏览器文档是HTML5版本编写。<!DOCTYPE>`不是 HTML 标签, 只是文档类型声明。

  • <html lang="en">用来定义当前文档的语言是英文，“zh-CN"是中文。

  • <meta charset="UTF-8">如果不写就会乱码，其中UTF-8被称为万国码，charset字符集是多个字符的集合，以便计算机能够识别和存储各种文字。

  • 第5、6行用于移动端的显示。

标题

标题（Heading）是通过 <h1> - <h6> 等标签进行定义的。

<h1> 定义最大的标题,<h6> 定义最小的标题。

<h1>This is a heading</h1>
<h2>This is a heading</h2>
<h3>This is a heading</h3>

段落

段落是通过 <p> 标签定义的。

<p>This is a paragraph</p>
<p>This is another paragraph</p>
  • 文本在一个段落中会根据浏览器窗口的大小自动换行。
  • 浏览器会自动地在段落的前后添加空行。

换行

如果您希望在不产生一个新段落的情况下进行换行（新行），请使用 <br />标签：

<p>This is<br />a para<br />graph with line breaks</p>

<br /> 元素是一个空的 HTML 元素。由于关闭标签没有任何意义，因此它没有结束标签。

即使 <br> 在所有浏览器中的显示都没有问题，使用 <br /> 是更长远的保障。

文本格式化

HTML 可定义很多供格式化输出的元素，比如粗体和斜体字。

标签描述
<b></b>或<strong></strong>加粗
<i></i>或<em></em>倾斜
<del></del>删除线
<ins></ins>下划线

容器标签

<div>和<span>是没有语义的，它们就是一个盒子，用来装内容的。

注释：<div> 元素常用作布局工具，因为能够轻松地通过 CSS 对其进行定位。

<div>这是头部</div>
<span>今日价格</span>
  • <div>标签用来布局，但是现在一行只能放一个<div>。大盒子
  • <span>标签用来布局，一行上可以多个<span>。小盒子

图像标签

在 HTML 中，图像由 <img> 标签定义，它并不会在网页中插入图像，而是从网页上链接图像。

<img>是空标签，只包含属性，并且没有闭合标签。

<img src="/i/eg_tulip.jpg"  alt="上海鲜花港 - 郁金香" />
属性值描述
alttext必需属性，规定图像的替代文本。
srcURL必需属性，规定显示图像的 URL。
heightpx(pixels)或%定义图像的高度（像素或百分比）。
widthpx(pixels)或%设置图像的宽度（像素或百分比）。

超链接

<a> 标签用于定义超链接，超链接可以让用户从一个网页跳转到另一个网页。

<a> 标签最重要的属性是 href 属性，它指定了链接目标的 URL。

<a href="http://www.w3school.com.cn/" target="_blank">Visit W3School!</a>
<p>如果把链接的 target 属性设置为 "_blank"，该链接会在新窗口中打开。</p>
  • 外部链接<a href="https://www.w3school.com.cn/">W3School</a>
  • 内部链接<a href="../index.html">首页</a>
  • 空链接<a href="#">空链接</a>
  • 下载链接<a href="xxx.zip">下载文件</a>
  • 锚点链接：点击链接可以快速定位到页面的某个位置
    • 在链接文本的href属性中，设置属性值为#名字的形式，如<a href="#two">第二集</a>
    • 找到目标位置标签，添加一个id属性，如<h3 id="two">第二集介绍</a>

表格标签

表格由 <table> 标签来定义，主要用于显示、展示数据。

每个表格均有若干行（由 <tr> 标签定义），每行被分割为若干单元格（由 <td> 标签定义）。字母 td 指表格数据（table data），即数据单元格的内容。数据单元格可以包含文本、图片、段落、表格等等。

表格的表头使用 <th>标签进行定义，大多数浏览器会把表头显示为粗体居中的文本。

<table border="1">
<tr>
<th>Heading</th>
<th>Another Heading</th>
</tr>
<tr>
<td>row 1, cell 1</td>
<td>row 1, cell 2</td>
</tr>
<tr>
<td>row 2, cell 1</td>
<td>row 2, cell 2</td>
</tr>
</table>

由于表格可能很长，为了更好的表示表格的语义，可以使用<thead> 标签定义表格的表头内容，<tbody> 标签定义表格的主体内容，<tfoot> 标签定义表格的表注内容（脚注或表注）。

合并单元格

使用colspan和rowspan属性跨行或跨列

<table border="1" width="500" height="249" cellspacing="0">
<tr>
	<td></td>
	<td colspan="2"></td>
</tr>
<tr>
	<td rowspan="2"></td>
	<td></td>
	<td></td>
</tr>
<tr>
	<td></td>
    <td></td>
</tr>
</table>

列表标签

HTML 支持有序、无序和自定义列表，列表是用来布局的。

无序列表始于 <ul> 标签。每个列表项始于 <li>。使用较多。

有序列表始于 <ol> 标签。每个列表项始于 <li> 。使用较少。

自定义列表以 <dl> 标签开始。每个自定义列表项以 <dt> 开始。每个自定义列表项的定义以 <dd> 开始。

<ul>
	<li>Coffee</li>
	<li>Milk</li>
</ul>

表单标签

HTML 表单用于搜集不同类型的用户输入。

构成标签说明
表单域<form>把它范围内的表单元素信息提交给服务器
表单元素<input>输入，最重要的表单元素
<select>下拉列表
<textarea>文本域
提示信息给用户输入每一项的提示信息

<hr /> 标签在 HTML 页面中创建水平线。hr 元素可用于分隔内容。

引用

HTML <q> 元素定义短的引用，<blockquote>元素定义被引用的节。

iframe 用于在网页内显示网页。
