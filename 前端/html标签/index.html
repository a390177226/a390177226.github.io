骨架标签

每个网页都会有的基本的结构标签。

标签名定义说明
<html></html>HTML标签页面中最大的标签，我们称为根标签
<head></head>文档的头部注意在head标签中我们必须要设置的标签是title
<title></title>文档的标题网页的标题
<body></body>文档的主体页面内容基本上是放在body里的

VSCode工具生成骨架标签代码

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

相较之前，主要多了：

  • <!DOCTYPE html>告诉浏览器文档是HTML5版本编写。<!DOCTYPE>`不是 HTML 标签, 只是文档类型声明。

  • <html lang="en">用来定义当前文档的语言是英文，“zh-CN"是中文。

  • <meta charset="UTF-8">如果不写就会乱码，其中UTF-8被称为万国码，charset字符集是多个字符的集合，以便计算机能够识别和存储各种文字。

  • 第5、6行用于移动端的显示。

标题

标题（Heading）是通过 <h1> - <h6> 等标签进行定义的。

<h1> 定义最大的标题,<h6> 定义最小的标题。

<h1>This is a heading</h1>
<h2>This is a heading</h2>
<h3>This is a heading</h3>

段落

段落是通过 <p> 标签定义的。

<p>This is a paragraph</p>
<p>This is another paragraph</p>
  • 文本在一个段落中会根据浏览器窗口的大小自动换行。
  • 浏览器会自动地在段落的前后添加空行。

换行

如果您希望在不产生一个新段落的情况下进行换行（新行），请使用 <br />标签：

<p>This is<br />a para<br />graph with line breaks</p>

<br /> 元素是一个空的 HTML 元素。由于关闭标签没有任何意义，因此它没有结束标签。

即使 <br> 在所有浏览器中的显示都没有问题，使用 <br /> 是更长远的保障。

水平线

<hr /> 标签在 HTML 页面中创建水平线。hr 元素可用于分隔内容。

<p>This is a paragraph</p>
<hr />
<p>This is a paragraph</p>
<hr />
<p>This is a paragraph</p>

文本格式化

HTML 可定义很多供格式化输出的元素，比如粗体和斜体字。

标签描述
<b></b>或<strong></strong>加粗
<i></i>或<em></em>倾斜
<del></del>删除线
<ins></ins>下划线

引用

HTML <q> 元素定义短的引用，<blockquote>元素定义被引用的节。

<!DOCTYPE html>
<html>
<body>

<p>WWF 的目标是：<q>构建人与自然和谐共存的世界。</q></p>
    
<p>浏览器通常会对 blockquote 元素进行缩进处理。</p>
<blockquote cite="http://www.worldwildlife.org/who/index.html">
五十年来，WWF 一直致力于保护自然界的未来。
WWF 工作于 100 个国家，并得到美国一百二十万会员及全球近五百万会员的支持。
</blockquote>

</body>
</html>
