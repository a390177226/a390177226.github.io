什么是 CSS？

  • CSS 指的是层叠样式表(Cascading Style Sheets)
  • CSS 描述了如何在屏幕、纸张或其他媒体上显示 HTML 元素
  • CSS 也是一种标记语言，用于定义网页的样式，包括针对不同设备和屏幕尺寸的设计和布局
  • CSS 节省了大量工作。它可以同时控制多张网页的布局
  • 外部样式表存储在 CSS 文件中，只需改一个文件即可更改整个网站的外观

CSS语法

CSS 语法由选择器和声明块组成：

  • 选择器指向需要设置样式的 HTML 元素
  • 声明块用花括号括起来，包含一条或多条用分号分隔的声明
  • 每条声明都包含一个 CSS 属性名称和一个值，以冒号分隔

如何添加 CSS

Inline CSS

行内样式（也称内联样式）可用于为单个元素应用唯一的样式。

特点：书写方便，权重高。

例子：<p style="color:red;">Test</p>

Internal CSS

如果一张 HTML 页面拥有唯一的样式，那么可以使用内部样式表。

内部样式是在 head 部分的 <style> 元素中进行定义。

<!DOCTYPE html>
<html>
<head>
<style>
body { background-color: linen;}
</style>
</head>
<body>
<h1>This is a heading</h1>
</body>
</html>

External CSS

通过使用外部样式表，只需修改一个文件即可改变整个网站的外观。

每张 HTML 页面必须在 head 部分的 <link> 元素内包含对外部样式表文件的引用。

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="mystyle.css">
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

外部样式表可以在任何文本编辑器中编写，并且必须以 .css 扩展名保存。上文中的"mystyle.css"：

body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}

Selectors

CSS 选择器用于“查找”（或选取）要设置样式的 HTML 元素。

我们可以将 CSS 选择器分为五类：

  • 简单选择器（根据名称、id、类来选取元素）
  • 组合器选择器（根据它们之间的特定关系来选取元素）
  • 伪类选择器（根据特定状态选取元素）
  • 伪元素选择器（选取元素的一部分并设置其样式）
  • 属性选择器（根据属性或属性值来选取元素）
Simple selectorsExampleExample description
#id#firstnameSelects the element with id=“firstname”
class.introSelects all elements with class=“intro”
element.classp.introSelects only <p> elements with class=“intro”
**Selects all elements
elementpSelects all <p> elements
element,element,…div, pSelects all <div> elements and all <p> elements

Values and Units

Length

CSS 有几种表示长度的不同单位。

许多 CSS 属性接受“长度”值，诸如 width、margin、padding、font-size 等。

长度是一个后面跟着长度单位的数字，诸如 10px、2em 等。

Colors

使用预定义的颜色名称或RGB、HEX、HSL、RGBA、HSLA值指定颜色。

设置文本的颜色：color: Tomato;

设置背景色：background-color: DodgerBlue;

Properties

Backgrounds

CSS 背景属性用于定义元素的背景效果。实际开发常见于logo或者一些装饰性的小图片或者是超大的背景图片或者是精灵图，优点是便于控制位置。

属性描述
background背景的简写属性
background-color设置元素的背景色
background-image设置元素的背景图像
background-repeat设置背景图像是否及如何重复
background-position设置背景图像的开始位置，最重要的属性
background-attachment设置背景图像是固定的还是与页面的其余部分一起滚动
背景半透明background: rgba(0,0,0,0.3);

Font

属性描述
font字体的简写属性
font-family规定文本的字体系列（字体族）
font-size规定文本的字体大小
font-style规定文本的字体样式
font-variant规定是否以小型大写字母的字体显示文本
font-weight规定字体的粗细

Text

CSS使用某些文本格式属性来设置样式。

属性描述注意点
color文本颜色通常用16进制
text-align文本水平对齐方式可以左对齐或右对齐，或居中对齐
text-indent文本缩进段落首行缩进两个字: text-indent=2em;
text-decoration文本装饰text-decoration: none; 通常用于从链接上删除下划线
line-height行高通常设置成元素的高度，使文本垂直居中

Borders

border属性允许您指定元素边框的样式、宽度和颜色。

PropertyDescription
border简写属性，border: border-width border-style(必须) border-color;
border-width简写属性，设置四条边框的宽度
border-style简写属性，设置四条边框的样式，常用的值有：solid、dashed、none
border-color简写属性，设置四条边框的颜色
border-radius简写属性，可设置圆角

在CSS中，也用于指定每个边框（top, right, bottom, and left）的属性，例如：border-top-style: dotted;

Margins and Padding

margin 属性用于在任何定义的边框之外，为元素周围创建空间。

padding 属性用于在任何定义的边界内的元素内容周围生成空间。

margin和padding都可以单独指定每一条边（top, right, bottom, and left）的属性。

例子：

设置上外边距是 25px，右外边距是 50px，下外边距是 75px，左外边距是 100px

设置上内边距是 25px，右内边距是 50px，下内边距是 75px，左内边距是 100px

p {
  margin: 25px 50px 75px 100px;
  padding: 25px 50px 75px 100px;
}

注意：margin: auto;可以使元素在其容器中水平居中

Height and Width

height 和 width 属性用于设置元素的高度和宽度。

Note: Remember that the height and width properties do not include padding, borders, or margins! They set the height/width of the area inside the padding, border, and margin of the element!

Box Model

盒模型实质上是一个包围每个 HTML 元素的盒子。它包括：外边距、边框、内边距以及实际的内容。

标准盒模型

在标准模型中，如果你给盒设置 width 和 height，实际设置的是 content。

padding 和 border 再加上设置的宽高一起决定整个盒子的大小。

IE盒模型

通过设置box-sizing: border-box;使用IE盒模型，所有宽度都是可见宽度，所以内容宽度是该宽度减去边框和填充部分。
