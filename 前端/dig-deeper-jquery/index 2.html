简介

  • jQuery 是一个轻量级的 “写的少，做的多” 的 JavaScript 库
  • jQuery 极大地简化了 JavaScript 编程，优化了DOM操作、事件处理、动画设计和Ajax交互
  • jQuery 很容易学习，提高开发效率

优点：

  • 轻量级
  • 跨浏览器兼容
  • 免费、开源

安装和使用

下载使用（推荐）

  1. 从 jquery.com 下载 jQuery 库
  2. 在工作目录下保存为jquery-3.6.0.min.js
  3. 引用jQuery：<script src="jquery-2.2.4.min.js"></script>

不下载，通过CDN（Content Delivery Network）使用

简单的例子

<!DOCTYPE html>
<head>
<meta charset="utf-8">
<script src="jquery-3.6.0.min.js"></script>
</head>

<body>
<p>aaaaaa</p>

<script>$('p').hide()</script>
</body>

jQuery 语法

jQuery 通过选取 HTML 元素，并对选取的元素执行某些操作

基础语法：$( selector ).action()

  • 美元符号$是jQuery 的别名

  • 选择符（selector）“查询” 和 “查找” HTML 元素

  • jQuery 的 action() 执行对元素的操作

jQuery 选择器

jQuery 选择器可以对 HTML 元素组或单个元素进行操作

选择器 描述 例子
元素选择器 基于元素名选取元素 选取所有 <p> 元素：$("p")
#id 选择器 通过 HTML 元素的 id 属性选取指定的元素 $("#test")
.class 选择器 通过指定的 class 查找元素 $(".test")
属性选择器 选取带有 href 属性的元素 $("[href]")
属性值选择器 选取所有 target 属性值等于 “_blank” 的 <a> 元素 $("a[target='_blank']")

jQuery 事件

jQuery 是为 HTML DOM 事件处理特别设计的

常见 HTML DOM 事件

鼠标事件 键盘事件 表单事件 文档/窗口事件
click keypress submit load
dblclick keydown change resize
mouseenter keyup focus scroll
mouseleave blur unload

jQuery 事件方法语法

在 jQuery 中，大多数 DOM 事件都有一个等效的 jQuery 方法

比如页面中指定一个点击事件

$("p").click(function(){
    // 动作触发后执行的代码!!
});

jQuery 效果和动画

隐藏和显示

方法 描述
hide() 隐藏 HTML 元素
show() 显示 HTML 元素
toggle() 切换hide()和show()方法 
<!DOCTYPE html>

<head>
    <meta charset="utf-8">
    <script src="jquery-3.6.0.min.js"></script>
</head>

<body>
    <p>123456789</p>
    <button id="toggle">toggle</button>
    <button id="hide">hide</button>
    <button id="show">show</button>

    <script>
        $("#hide").click(function () {
            $("p").hide();
        });
        $("#show").click(function () {
            $("p").show();
        });
        $("#toggle").click(function () {
            $("p").toggle();
        });
    </script>
</body>

淡入淡出

jQuery 提供四种实现元素的淡入淡出效果的方法

  • fadeIn()：淡入已隐藏的元素
  • fadeOut()：淡出可见元素
  • fadeToggle()：在 fadeIn() 与 fadeOut()方法之间进行切换
  • fadeTo()：渐变为给定的不透明度 ( 值介于 0 与 1 之间 )

滑动

jQuery 提供了三个方法来创建元素的滑动效果

  • slideDown()：向下滑动元素
  • slideUp()：向上滑动元素
  • slideToggle()：在 slideDown() 与 slideUp() 方法之间进行切换

动画和停止动画

jQuery 中的animate() 函数可以创建自定义动画

jQuery 中的stop() 方法用于停止动画或效果，包括滑动、淡入淡出和自定义动画

jQuery 方法链 ( chaining )

jQuery 还提供了一种一种名为链接（chaining）的技术，允许我们在相同的元素上运行多条 jQuery 命令，这样做的好处就是省去了多次查找相同的元素

下面的范例把 css()、slideUp() 和 slideDown() 链接在一起，“p1” 元素首先会变为红色，然后向上滑动，再然后向下滑动

<!DOCTYPE html>

<head>
    <meta charset="utf-8">
    <script src="jquery-3.6.0.min.js"></script>
</head>

<body>
    <p id="p1">zzb</p>
    <button>点我</button>

    <script>
        $("button").click(function () {
            $("#p1").css("color", "red").slideUp(2000).slideDown(2000);
        });
    </script>
</body>

jQuery DOM 操作

jQuery 中非常重要的部分，就是操作 DOM 的能力。

jQuery 提供一系列与 DOM 相关的方法，这使访问和操作元素和属性变得很容易。

获取内容和属性

方法 描述 例子
text() 获取所选元素的文本内容 $("#test").text()
html() 获取所选元素的内容（包括 HTML 标记） $("#test").html()
val() 获取表单字段的值 $("#test").val()
attr() 获取属性值 $("#twle").attr("href")

设置内容和属性

方法 描述 例子
text() 设置所选元素的文本内容 $("#test1").text("Hello world!")
html() 设置所选元素的内容（包括 HTML 标记） $("#test2").html("<b>Hello world!</b>")
val() 设置表单字段的值 $("#test3").val("TWLE")
attr() 设置属性值 $("#twle").attr("href","http://www.twle.cn/jquery")

添加元素

jQuery 提供了四个简单使用的方法用于添加新内容或元素

  1. append() - 在被选元素内部结尾插入内容
  2. prepend() - 在被选元素内部开头插入内容
  3. after() - 在被选元素外部之后插入内容
  4. before() - 在被选元素外部之前插入内容

删除元素

如果想要删除元素和内容，可以使用下面的 jQuery 方法

  1. remove() - 删除被选元素及其子元素
  2. empty() - 从被选元素中删除子元素

获取并设置 CSS 类

jQuery 拥有若干进行 CSS 操作的方法

  1. addClass() - 向被选元素添加一个或多个类
  2. removeClass() - 从被选元素删除一个或多个类
  3. toggleClass() - 对被选元素进行添加/删除类的切换操作
  4. css() - 设置或返回样式属性

jQuery 尺寸

jQuery 提供了六个方法用来设置或返回元素和浏览器窗口的尺寸

  1. width()
  2. height()
  3. innerWidth()
  4. innerHeight()
  5. outerWidth()
  6. outerHeight()

jQuery 遍历

向上遍历 DOM 树

jQuery 提供了下面三个方法用于向上遍历 DOM 树

  1. parent()
  2. parents()
  3. parentsUntil()

向下遍历 DOM 树

jQuery 提供了下面两个用于向下遍历 DOM 树

  1. children() - 返回被选元素的所有直接子元素
  2. find() - 返回被选元素的后代元素，一路向下直到最后一个后代

jQuery AJAX

jQuery 提供多个与 AJAX 有关的方法

通过 jQuery AJAX 方法，可以使用 HTTP Get 和 HTTP Post 从远程服务器上请求文本、HTML、XML 或 JSON，而且还可以把这些外部数据直接载入网页的被选元素中

jQuery 解放了 AjAx

  • 编写常规的 AJAX 代码并不容易，因为不同的浏览器对 AJAX 的实现并不相同，这意味着您必须编写额外的代码对浏览器进行测试
  • jQuery 团队为我们解决了这个难题，我们只需要一行简单的代码，就可以实现 AJAX 功能

load() 方法

jQuery load() 方法可以从服务器加载数据，并把返回的数据放入被选元素中

基本语法：$(selector).load(URL,data,callback);

  • 必需的 URL 参数设置了希望加载的 URL
  • 可选的 data 参数规定与请求一同发送的查询字符串键/值对集合
  • 可选的 callback 参数是 load() 方法完成后所执行的回调函数

假设服务器上有一个文件 test.txt，内容如下

<h4>jQuery AJAX 是个非常棒的功能</h4>
<p>这是段落的一些文本</p>
<p>简单教程，简单编程</p>

下面的范例会把文件 “test.txt” 的内容加载到指定的 <div> 元素中

<!DOCTYPE html>

<head>
    <meta charset="utf-8">
    <script src="jquery-3.6.0.min.js"></script>
</head>

<body>
    <div id="div1">
        <h2>使用 jQuery AJAX 修改文本内容</h2>
    </div>
    <button>获取外部内容</button>

    <script>
        $("button").click(function () {
            $("#div1").load("./test.txt");
        });
    </script>
</body>

get() 和 post() 方法

jQuery get() 和 post() 方法用于通过 HTTP GET 或 POST 请求从服务器请求数据

  • $.get(URL,callback)

    • 必需的 URL 参数用于设置希望请求的 URL
    • 可选的 callback 参数是请求成功后所执行的回调函数

  • $.post( URL,data,callback )

    • 可选的 data 参数规定连同请求发送的数据

jQuery noConflict()方法

noConflict() 用于处理和其它 JavaScript 框架 $ 符号冲突的情况

noConflict() 方法会释放对 $ 标识符的控制，这样其它脚本就可以使用它了

// 创建自己的简写
var jq = $.noConflict();
jq(document).ready(function(){
  jq("button").click(function(){
    jq("p").text("jQuery 仍然在工作!");
  });
});
