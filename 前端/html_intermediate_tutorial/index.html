Block and Inline Elements

每个HTML元素都有一个默认的显示值，这取决于它是什么类型的元素。

Display value区别例子
Block-level elements总是从新行开始，占据可用的全部宽度，有上下外边距（margin）<p>
Inline elements不会从新行开始，只占用所需的宽度，没有上下外边距（margin）<a>

注意：行内元素不能包含块级元素！

Div and Span

大多数HTML 标签都有语义（例如，p表示段落、h1表示标题），但是span和div标签没有特定的语义。

<div> 元素是块级元素，通常是其他 HTML 元素的容器，另一个常见的用途是文档布局。

<span> 元素是行内元素，可用作文本的容器。

<div id="scissors">
    <p>This is <span class="paper">crazy</span></p>
</div>

注：HTML5引入了许多新的语义标签，请尽可能的使用语义标签来代替<div>和<span>。

Head Element

HTML <head>元素是以下元素的容器：<title>、<style>、<meta>、<link>、<script>和<base>。

标签说明
<title>定义文档的标题，是必需的
<style>定义单个HTML页的样式信息
<link>定义当前文档和外部资源之间的关系，最常用于链接外部样式表
<meta>通常用于指定字符集、页面描述、关键字、文档作者和视口设置。
<script>用于定义客户端JavaScripts
<base>为页面上的所有链接规定默认地址或默认目标（target）

Meta Tags

元数据（metadata）是关于数据的信息。

<meta>标签提供关于 HTML 文档的元数据。元数据不会显示在页面上，但是对于机器是可读的。

常用说明
<meta charset="UTF-8">指定HTML文档的字符编码为UTF-8
<meta name="keywords" content="HTML, CSS, JS">为搜索引擎定义关键字
<meta name="description" content="Tutorials for HTML">定义网页的描述
<meta name="author" content="John Doe">定义网页的作者
<meta http-equiv="refresh" content="30">每30秒刷新一次网页
<meta name="viewport" content="width=device-width, initial-scale=1.0">设置视口，兼容所有设备
<meta http-equiv="X-UA-Compatible" content="IE=edge">告诉IE浏览器使用渲染引擎（edge）

Layout Elements in HTML5

HTML5提供了一些新的语义元素，可用于定义网页的不同部分：

常用说明例子
<header>被用作介绍性内容的容器或一组导航链接通常包含标题、logo、作者信息
<nav>定义导航链接集合通常是大型的导航链接块
<section>定义文档中的节有主题的内容组，通常具有标题
<article>指定独立的、自包含的内容论坛帖子、博客文章、报纸文章
<aside>页面主内容之外的某些内容旁白内容应与周围内容间接相关
<footer>为文档或节规定页脚页脚通常包含文档的作者、版权信息、使用条款链接、联系信息等
<main>规定文档的主内容，是独一无二的不包含任何在一系列文档中重复的内容，比如导航栏链接(nav)

HTML Global Attributes

全局属性是所有 HTML 元素都可以使用的属性。

核心属性说明用法
id规定元素的唯一的id链接锚（link anchor），通过 JS（HTML DOM）或通过 CSS 为带有指定 id 的元素改变或添加样式
class规定元素的类名（classname）大多数时候用于指向样式表中的类（class），也可以利用它通过 JS来改变带有指定 class 的 HTML 元素
style规定元素的行内样式（inline style）style 属性将覆盖任何全局的样式设定
title规定关于元素的额外信息这些信息通常会在鼠标移到元素上时显示一段工具提示文本（tooltip text）
其他属性说明用法
lang规定元素内容的语言<html lang="en">指定英文，zh为中文
accesskey指定激活/聚焦元素的快捷键使用 [ALT] + accesskey
tabindex规定元素的 tab 键控制次序<a href="#" tabindex="1">Google</a>从1开始
dir规定元素内容的文本方向ltr默认，rtl从右向左的文本方向
HTML5新增说明用法
contenteditable规定元素内容是否可编辑<p contenteditable="true">Test</p>
draggable规定元素是否可拖动<p draggable="true">Test</p>
hidden浏览器不应显示已规定 hidden 属性的元素<p hidden>This paragraph should be hidden.</p>
data-*自定义属性，嵌入自定义数据<p data-animal-type="bird">Owl</p>

Multimedia and Embedding

Video and Audio

Web 上的多媒体指的是声音、音乐、视频和动画。

HTML <video>元素用于在网页上显示Ogg、MP4、WebM格式的视频。

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>

  • controls属性添加视频控件，如播放、暂停和音量

  • <source>元素允许指定浏览器可以从中选择的备选视频文件，浏览器将使用第一个识别的格式

  • <video>和</video>之间的文本将仅在不支持<video>元素的浏览器中显示

HTML <audio>元素用于在网页上播放MP3, WAV, OGG格式的音频文件。

<audio controls autoplay muted>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

Embedding

嵌入（Embedding）指的是让你在网页中嵌入各种类型的内容。

主要的嵌入方法是使用<iframe>、<embed>和<object>元素。

<iframe>用来嵌入其他网页，通常是嵌入第三方内容（例如YouTube视频和地图）。

<object>和<embed>元素是用来嵌入多种类型的外部内容的通用嵌入工具，其中包括像Java小程序和Flash，PDF这样的插件技术，甚至像视频，SVG和图像的内容！

注：实际上，这些元素并不常用

  • Applet几年来一直没有被使用
  • 由于许多原因，Flash不再受欢迎
  • PDF更倾向于被链接而不是被嵌入
  • 其他内容，如图像和视频都有更优秀、更容易元素来处理
